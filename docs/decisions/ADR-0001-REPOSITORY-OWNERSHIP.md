# ADR-0001: Repository Ownership

## Status
Accepted

## Decision
Master-All-Strings is the product-owning repository for the new Master Producer and related musical-runtime technology.

## Rejected ownership models

- Master-Producer legacy archive
- luthiers-toolbox
- CNC-Production-Shop

## Consequences

Modern implementation work happens in this repository, while legacy repositories remain evidence sources or future integration partners.
